About Manicule

This website serves as a demo of the Manicule application version 3 for exploring unique books and manuscripts. This data is stored as a VisColl model in JSON format, as generated by the VCEditor tool.

It is intended to either used as a standalone website, or as a JavaScript library embedded in another website. It uses web components, an open standard that makes it possible to use custom HTML tags to add functionality.

Basic usage of Manicule is demonstrated in the examples on this page:

  1. Add a folder under the data/ folder in this application.
  2. The folder name should match the filename of the VisColl JSON file.

Add the following HTML tags wherever you'd like all or part of the manuscript navigation to appear: 

          <collation-model id="62782273d3ef8000013aba57"></collation-model>

Inside the collation-model tag, add any of the custom elements provided by Manicule to show spreads, individual leaves, thumbnails, or a complete tour. Use View Source on your browser to see how these examples are described.
