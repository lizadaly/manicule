Manicule

This website serves as a demo of the Manicule web app version 3 for exploring unique books and manuscripts. The Browse view allows you to read the fragments in order, following the modern pagination on the leaves and bifolia. Marginal notes beside each facsimile page indicate which lines from The Canterbury Tales are present; these notes can be changed by altering the description attribute in pages.json. Color-coded categories indicate which tale is present; these categories can also be changed in pages.json and the colors updated in metadata.ts. The Structure view shows where these fragments are conjoined, or not. This data is stored in structure.xml, which visualizes the format of a book using the VisColl data model.
